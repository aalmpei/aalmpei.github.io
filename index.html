Remote educational resource MPEI PROCESSOR

supports the theoretical and practical study of the properties and algorithms of finite groups, rings, fields, and based on these algebraic structures of information security solutions, in particular, cryptographic systems and protocols. It has a wide range of algebraic structures studied and allows to perform calculations using compiled interactively and programs based on MPEI AAL library

Welcome to AAL library demonstration.

Here is the interactive python console with AAL library imported. You can find out how it works in the repository of this project and in repository of the library
